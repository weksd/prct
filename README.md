# Описание
Данный веб-сайт был разработан для онлайн-записи в салон красоты "MatrixGlam". Приложение позволяет клиентам удобно записываться на услуги салона через интернет, выбирать мастера и удобное время, а администраторам — управлять всеми записями через защищённую панель управления. Сайт представляет собой полноценную систему бронирования с интуитивно понятным интерфейсом, работающую непосредственно в браузере без необходимости установки дополнительного программного обеспечения.
# Установка
Для использования сайта достаточно открыть файл index.html в любом современном браузере — приложение работает локально на компьютере пользователя и не требует подключения к интернету или серверу. Все данные хранятся в локальном хранилище браузера, что обеспечивает быстрый доступ и сохранность информации. Сайт имеет адаптивный дизайн, корректно отображается на компьютерах, планшетах и мобильных устройствах.
# Использование
Приложение позволяет клиентам выбирать из трёх основных услуг салона: женская стрижка, мужская стрижка и маникюр. Для каждой услуги представлен соответствующий мастер с подробным описанием его квалификации и опыта. После выбора мастера пользователь заполняет простую форму записи, указывая свои контактные данные, дату и удобное время посещения. Все созданные записи автоматически сохраняются в системе. Администраторы имеют доступ к защищённой панели управления, где могут просматривать все поступившие заявки, подтверждать записи или удалять их при необходимости. Панель администратора защищена системой авторизации с логином и паролем, что обеспечивает конфиденциальность данных клиентов.
